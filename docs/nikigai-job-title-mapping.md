# Nikigai Job Title Mapping

Comprehensive mapping of role archetypes to job titles across industries, employment types, and career stages.

---

## How to Use This Mapping

After skill clusters are identified and role archetypes are assigned, this mapping helps suggest:
- **Job titles** aligned with the user's path type (Career vs. Entrepreneurial)
- **Industries** where those skills are valued
- **Career stages** from entry to executive
- **Employment types** from corporate to freelance

## How to Use This Mapping

### Implementation in Skills Lens

```javascript
function suggestJobTitles(roleArchetype, userPreferences) {
  const { industry, pathType, careerStage } = userPreferences

  // Get job title mapping for this archetype
  const mapping = jobTitleMappings[roleArchetype]

  // Filter by industry if specified
  let titles = industry
    ? mapping.jobTitles[industry]
    : Object.values(mapping.jobTitles).flat()

  // Filter by path type (career vs entrepreneurial)
  if (pathType === 'Entrepreneurial Opportunity') {
    titles = mapping.entrepreneurialPaths
  }

  // Filter by career stage if specified
  if (careerStage) {
    titles = titles.filter(t => t.stages.includes(careerStage))
  }

  return titles.slice(0, 10) // Top 10 suggestions
}
```

## Career Stage Definitions

- **Entry:** 0-3 years experience, early career
- **Mid:** 3-7 years experience, established professional
- **Senior:** 7-15 years experience, expert level
- **Executive:** 15+ years experience, C-suite or Director level
